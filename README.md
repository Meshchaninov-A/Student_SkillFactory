# Student_SkillFactory
##	Описание
У студента есть следующие важные свойства:
 * имя
 * фамилия
 * возраст
 * наличие опыта
 * цель обучения
 * номер группы
 * количество выполненных заданий
 * количество пройденных модулей  
 
Можно выделить следующие методы:
* выполнить задание
* задать вопрос
* пройти модуль
* пообщаться с куратором
